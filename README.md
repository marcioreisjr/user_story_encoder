# User Story Encoder

A tool to create/encode standardized user stories in JSON format.

## Features

- **Dynamic Form**: Easily add/remove acceptance criteria and dependencies.
- **Real-time JSON Preview**: See the data structure as you type.
- **Download JSON**: Export your user story to a `.json` file.
- **Dark Mode Design**: Sleek dark blue and white palette.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React
- Vite
- Vanilla CSS (Dark Theme)
